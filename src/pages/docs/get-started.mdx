### 快速开始

AuthHub 提供 &zwnj;**身份验证即服务**&zwnj;，支持快速集成到任何 Web 应用。根据项目类型选择部署方式：

- &zwnj;**新建项目**&zwnj;：还没有代码的新起项目，将 AuthHub 作为脚手架使用
- &zwnj;**已有项目**&zwnj;：现有代码库中集成 AuthHub

---

### 前提条件

- Node.js 16+ / Python 3.8+（根据项目语言）
- 已注册 [AuthHub 控制台](https://console.authhub.com) 并获取 `CLIENT_ID` 和 `API_KEY`
- 终端访问权限

---

### 新项目部署

#### 1. 创建项目

```bash
# React 示例
npx create-react-app my-auth-app
cd my-auth-app
```

#### 2. 安装 AuthHub SDK

```bash
npm install @authhub/web-sdk
```

或

```bash
yarn add @authhub/web-sdk
```

#### 3. 初始化配置

在项目根目录创建 .env 文件：

```ini
AUTHHUB_CLIENT_ID=your_client_id
AUTHHUB_API_KEY=your_api_key
```

#### 4. 集成到入口文件

```jsx
import { AuthHubProvider } from "@authhub/web-sdk";

ReactDOM.render(
  <AuthHubProvider>
    <App />
  </AuthHubProvider>,
  document.getElementById("root")
);
```

#### 5. 启动应用

```bash
npm start
```
